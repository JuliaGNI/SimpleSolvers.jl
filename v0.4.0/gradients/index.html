Gradients

The supertype Gradient comprises different ways of taking gradients:

  • GradientFunction,
  • GradientAutodiff,
  • GradientFiniteDifferences.

We first start by showing GradientAutodiff:

f(x::AbstractArray) = sum(x .^ 2)
x = rand(3)
grad = GradientAutodiff(f, x)
GradientAutodiff{Float64, typeof(Main.f), ForwardDiff.GradientConfig{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3, Vector{ForwardDiff.Dual{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3}}}}(Main.f, ForwardDiff.GradientConfig{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3, Vector{ForwardDiff.Dual{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3}}}((Partials(1.0, 0.0, 0.0), Partials(0.0, 1.0, 0.0), Partials(0.0, 0.0, 1.0)), ForwardDiff.Dual{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3}[Dual{ForwardDiff.Tag{typeof(Main.f), Float64}}(6.9260655835124e-310,6.92606558351713e-310,6.9260655835203e-310,6.92606558352345e-310), Dual{ForwardDiff.Tag{typeof(Main.f), Float64}}(6.9260655835266e-310,6.9260655835298e-310,6.92606558353294e-310,6.9260655835361e-310), Dual{ForwardDiff.Tag{typeof(Main.f), Float64}}(6.92606558353926e-310,6.92606558354242e-310,6.9260655835456e-310,6.92606558354875e-310)]))

Instead of calling GradientAutodiff(f, x) we can equivalently do:

grad = Gradient(f, x; mode = :autodiff)
GradientAutodiff{Float64, typeof(Main.f), ForwardDiff.GradientConfig{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3, Vector{ForwardDiff.Dual{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3}}}}(Main.f, ForwardDiff.GradientConfig{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3, Vector{ForwardDiff.Dual{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3}}}((Partials(1.0, 0.0, 0.0), Partials(0.0, 1.0, 0.0), Partials(0.0, 0.0, 1.0)), ForwardDiff.Dual{ForwardDiff.Tag{typeof(Main.f), Float64}, Float64, 3}[Dual{ForwardDiff.Tag{typeof(Main.f), Float64}}(6.9260554558433e-310,0.0,6.92610012208755e-310,0.0), Dual{ForwardDiff.Tag{typeof(Main.f), Float64}}(6.92609295353495e-310,0.0,6.92609570562684e-310,0.0), Dual{ForwardDiff.Tag{typeof(Main.f), Float64}}(6.9260554558678e-310,0.0,6.9260957051818e-310,0.0)]))

When calling an instance of Gradient we can use the functions gradient and gradient![1]:

gradient(x, grad)
