f = x -> tanh.(x)
nls = NonlinearSystem(f, x)

We now create an instance of NewtonSolver which also allocates a SimpleSolvers.NonlinearSolverStatus:

solver = NewtonSolver(x, f)
i=   0,
x= NaN,
f= NaN,
rxₐ= NaN,
rfₐ= NaN

Note that all variables are initialized with NaNs.

For the first step we therefore have to call update![1]:

update!(solver, x)
i=   1,
x=3.000000e+00,
f=9.950548e-01,
rxₐ= NaN,
rfₐ=1.316321e+00

Note that the residuals are still NaNs however as we need to perform at least two updates in order to compute them. As a next step we write:

x = [2., 1.2]
update!(solver, x)
i=   2,
x=2.000000e+00,
f=9.640276e-01,
rxₐ=1.004988e+00,
rfₐ=1.274492e+00
